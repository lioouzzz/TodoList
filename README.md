### 介紹
https://mytodolist-hedre0bgcyb8dkgk.canadacentral-01.azurewebsites.net/  
一個基於 C# MVC 架構的待辦事項列表應用程式，提供直觀的任務管理功能。
ToDoList 採用 ASP.NET MVC 架構開發，能夠有效管理日常待辦事項、利用會員系統管理權限。  
應用程式前端利用 Tailwind CSS CDN 美化界面，後端資料庫部分支援本機 MS SQL 與 Azure 雲端資料庫，並已部署在 Azure F1 免費方案中，提供穩定且免費的雲端服務。

### 技術棧
- **後端**： C# MVC (ASP.NET MVC)
- **前端**： HTML、CSS、JavaScript，使用 Tailwind CSS CDN
- **資料庫**： 本機 MS SQL 與 Azure 雲端資料庫
- **部署**： 部署於 Azure F1 免費方案 (Azure App Service)
